# hy · 경력입사 전략 & 보상 대시보드

반도체 경력 채용 지원 전략과 보상/복지 분석을 정리한 단일 페이지 대시보드입니다.
회사명은 `hy`로 마스킹 표기합니다.

## 보기

- 로컬: `index.html`을 브라우저로 열기
- GitHub Pages: `Settings → Pages → Branch: main / root` 설정 후
  `https://heekeunlee.github.io/hy/` 접속

## 구성

1. **① 경력입사 전략** — 포지셔닝, 직무 적합도 랭킹, 두 갈래 전략(합격 우선 / 워라밸), 직무 선택 결정 질문, 직무별 핵심 메시지, 자소서 키워드 체크
2. **② 복지 · 급여 혜택** — 보상 구조(기본급·PS·PI), 직급별 연봉 추정, 복리후생, 지원 전략과의 연결

## 주의

- 보상 수치(연봉/성과급)는 언론·커뮤니티 **추정치**이며 공식 확정값이 아닙니다.
- 복지 항목은 공식 채용 페이지 기준입니다.
- 개인 의사결정 참고용 문서입니다.
